"인공지능(Artificial Intelligence)", "머신러닝(Machine Learning)", "딥러닝(Deep Learning)" 같은 단어들은 이제 일상 속 어디에서나 들을 수 있습니다. 그런데 이 거대한 기술은 어떤 원리로 '생각'하고 '판단'할 수 있게 된 걸까요? 그 시작은 1957년에 고안된, 인간의 뇌 신경 세포(뉴런)를 모방한 아주 단순한 알고리즘에서 출발합니다.

그 알고리즘이 바로 **퍼셉트론(Perceptron)**입니다.

이번 글에서는 딥러닝의 기원이 되는 퍼셉트론의 기본 개념을 알아보고, 이 단순한 모델이 어떻게 논리 회로를 구현하며, 또 어떤 한계점을 가졌고 어떻게 해결했는지 살펴봅니다.

---

## 🧠 뉴런을 흉내 낸 퍼셉트론

인간의 뇌는 **뉴런(Neuron)**이라는 신경 세포들로 이루어져 있습니다. 뉴런은 여러 곳에서 '입력 신호'를 받아, 이 신호들의 합이 특정 임곗값을 넘어서면 '출력 신호'를 다음 뉴런으로 전달합니다.

퍼셉트론은 이 구조를 그대로 수학적으로 모델링합니다.

### 퍼셉트론의 작동 원리

1. **입력(Input):** $x_1$, $x_2$와 같은 여러 개의 입력값을 받습니다.
2. **가중치(Weight):** 각각의 입력값에는 $w_1$, $w_2$처럼 고유한 중요도 역할을 하는 **'가중치'**가 곱해집니다. 가중치($w$)가 클수록 해당 입력 신호가 결과에 더 큰 영향을 미칩니다.
3. **합산 및 비교:** '입력 $\times$ 가중치'의 총합($w_1x_1 + w_2x_2$)을 구합니다.
4. **활성화(Activation):** 이 합이 정해진 **임곗값($\theta$)보다 크면 1**을 출력(신호 전달)하고, **작거나 같으면 0**을 출력(신호 무시)합니다.

이를 수식으로 표현하면 다음과 같습니다.

$$
y = \begin{cases} 
0 & (w_1x_1 + w_2x_2 \le \theta) \\
1 & (w_1x_1 + w_2x_2 > \theta)
\end{cases}
$$

위 수식에서 $\theta$를 이항하면 $w_1x_1 + w_2x_2 - \theta > 0$ 형태로 바꿀 수 있습니다. 이때 $-\theta$를 **'편향'**이라고 부르는데, 뉴런이 얼마나 쉽게 활성화되는지를 조절하는 기준점 역할을 합니다. 이 편향($b$)을 대입하면 퍼셉트론 수식을 다음과 같이 표현할 수 있습니다.

$$
y = \begin{cases} 
0 & (w_1x_1 + w_2x_2 + b \leq 0) \\
1 & (w_1x_1 + w_2x_2 + b > 0)
\end{cases}
$$

## ⚙️ 논리 게이트 만들기

이 단순한 모델을 통해 논리 회로의 기초인 **게이트(Gate)**를 구현할 수 있습니다.

### AND 게이트

AND 게이트는 두 개의 입력($A$, $B$)이 **모두 1일 때만** 1을 출력합니다.

| **$A$** | **$B$** | **AND** |
| :---: | :---: | :---: |
| **0** | **0** | **0** |
| **0** | <span class="text-green">**1**</span> | **0** |
| <span class="text-green">**1**</span> | **0** | **0** |
| <span class="text-green">**1**</span> | <span class="text-green">**1**</span> | <span class="text-green">**1**</span> |

이 AND 게이트를 퍼셉트론으로 구현할 수 있습니다. 가중치와 편향 값을 잘 설정하기만 하면 됩니다.

예를 들어, $w_1 = 0.5$, $w_2 = 0.5$, $\theta = 0.7$ (즉, $b = -0.7$)로 설정해 봅시다.

* (0, 0) → $\space (0.5 \times 0) + (0.5 \times 0) - 0.7 = -0.7 \space (\leq 0) \space$ → **0**
* (0, 1) → $\space (0.5 \times 0) + (0.5 \times 1) - 0.7 = -0.2 \space (\leq 0) \space$ → **0**
* (1, 0) → $\space (0.5 \times 1) + (0.5 \times 0) - 0.7 = -0.2 \space (\leq 0) \space$ → **0**
* (1, 1) → $\space (0.5 \times 1) + (0.5 \times 1) - 0.7 = +0.3 \space (\gt 0) \space$ → **1**

모든 경우에서 AND 게이트의 진리표와 정확히 일치하는 것을 볼 수 있습니다.

#### Python 코드로 구현하기

```python
import numpy as np

def AND(x1, x2):
    x = np.array([x1, x2])
    w = np.array([0.5, 0.5])  # 가중치
    b = -0.7                  # 편향 (임곗값 0.7)
    result = np.sum(w * x) + b

    if result <= 0:
        return 0
    else:
        return 1

# 결과 확인
print(f"(0, 0) -> {AND(0, 0)}")  # (0, 0) -> 0
print(f"(0, 1) -> {AND(0, 1)}")  # (0, 1) -> 0
print(f"(1, 0) -> {AND(1, 0)}")  # (1, 0) -> 0
print(f"(1, 1) -> {AND(1, 1)}")  # (1, 1) -> 1
```

### OR / NAND 게이트

같은 원리로, $w$와 $b$ 값만 조절하면 **OR 게이트**(두 입력 중 하나 이상 1일 때 1 출력)와 **NAND 게이트**(AND의 반대)도 쉽게 만들 수 있습니다.

* **OR 게이트:** $\space w = [0.5, 0.5]$, $\space b = -0.2$
* **NAND 게이트:** $\space w = [-0.5, -0.5]$, $\space b = 0.7$

## 💥 한계: XOR 문제

AND, OR, NAND 게이트는 모두 좌표 평면에서 '직선 하나'로 0과 1을 구분할 수 있는 **선형(Linear)** 문제입니다.

하지만 1969년, 마빈 민스키(Marvin Minsky)는 이 단순한 퍼셉트론(단층 퍼셉트론)에 치명적인 한계가 있음을 증명합니다. **XOR(Exclusive OR) 게이트** 문제를 풀 수 없다는 것입니다.

XOR 게이트는 두 입력이 **서로 다를 때만** 1을 출력합니다.

| **$A$** | **$B$** | **XOR** |
| :---: | :---: | :---: |
| **0** | **0** | **0** |
| **0** | <span class="text-green">**1**</span> | <span class="text-green">**1**</span> |
| <span class="text-green">**1**</span> | **0** | <span class="text-green">**1**</span> |
| <span class="text-green">**1**</span> | <span class="text-green">**1**</span> | **0** |

이 경우는 다른 게이트들과 달리 0과 1의 분포를 **직선 하나로** 분리할 방법이 전혀 없고, 이것이 **비선형(Non-linear)** 문제입니다. 이 문제로 인해 인공지능 연구는 첫 번째 겨울을 맞이하게 됩니다.

---

## 💡 해법: 다층 퍼셉트론(MLP)

겨울은 길었지만, 해법은 의외로 간단한 곳에 있었습니다.

> **"퍼셉트론 하나로 안 된다면, 여러 개를 쌓으면 어떨까?"**

이것이 바로 **다층 퍼셉트론(Multi-Layer Perceptron, MLP)**이며, 신경망(Neural Network)의 시작입니다.

XOR 게이트는 기존의 게이트들을 조합하여 만들 수 있습니다: `XOR(A, B) = AND(NAND(A, B), OR(A, B))`

이 논리 회로를 퍼셉트론으로 그대로 옮겨봅시다.

* **Layer 1:** 입력 $A$, $B$를 받아 각각 **NAND 값**(`s1`)과 **OR 값**(`s2`)을 계산합니다.
* **Layer 2:** Layer 1의 출력값 `s1`과 `s2`를 새로운 입력으로 받아 **AND 값**을 계산합니다.

#### Python 코드로 구현하기

```python
# AND, OR, NAND 함수는 앞서 정의했다고 가정하고 생략

def XOR(x1, x2):
    s1 = NAND(x1, x2)  # Layer 1
    s2 = OR(x1, x2)    # Layer 1
    y = AND(s1, s2)    # Layer 2
    return y

# 결과 확인
print(f"(0, 0) -> {XOR(0, 0)}")  # (0, 0) -> 0
print(f"(0, 1) -> {XOR(0, 1)}")  # (0, 1) -> 1
print(f"(1, 0) -> {XOR(1, 0)}")  # (1, 0) -> 1
print(f"(1, 1) -> {XOR(1, 1)}")  # (1, 1) -> 0
```

## 1부 끝

이번에는 딥러닝의 가장 기초가 되는 **퍼셉트론**에 관해 알아봤습니다.

* **퍼셉트론**은 인간의 뉴런을 모방한 단순 모델입니다.
* **단층 퍼셉트론**은 가중치와 편향을 이용해 선형 문제(AND, OR, NOT 게이트)를 해결할 수 있습니다. 하지만 **비선형** 문제(XOR 게이트)는 해결할 수 없다는 문제에 직면했습니다.
* 퍼셉트론을 여러 층으로 쌓은 **다층 퍼셉트론(MLP)**을 통해 이 문제를 해결할 수 있었고, 이것이 바로 딥러닝의 핵심인 **신경망(Neural Network)**의 시초입니다.
