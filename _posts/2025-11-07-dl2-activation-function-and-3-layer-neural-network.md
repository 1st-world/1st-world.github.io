하지만 퍼셉트론의 단순한 '0 또는 1' 스위치(계단 함수)만으로 실제 복잡한 문제들을 해결하기에는 한계가 극명합니다. 신경망이 더 똑똑하게 '생각'하려면 더 유연한 부품이 필요합니다.

이 글에서는 신경망의 핵심 부품인 **활성화 함수(Activation Function)**란 무엇이며, 왜 필요한지 알아봅니다. 그리고 NumPy를 이용해 3층 신경망의 **순전파(Forward Propagation)** 과정을 코드로 구현해 봅니다.

## 신경망의 '유연함'을 더하는 활성화 함수

**활성화 함수**는 뉴런이 받은 입력 신호의 총합을 다음 층으로 어떻게 출력할지 결정하는 함수입니다. [1부](/posts/dl1-what-is-a-perceptron/)의 퍼셉트론에서는 '계단 함수(Step Function)'를 사용했습니다.

그렇다면 왜 계단 함수 대신 다른 함수를 사용할까요?

가장 큰 이유는 **비선형성(Non-linearity)을 추가하기 위해서**입니다.

만약 활성화 함수가 $y=x$처럼 선형(Linear)이라면, 신경망의 층을 아무리 깊게 쌓아도 결국은 하나의 거대한 선형 함수가 될 뿐입니다. 이는 XOR 문제도 풀 수 없는 단층 퍼셉트론과 다를 게 없다는 거죠. 비선형 함수를 각 층에 적용해야만 신경망이 층을 쌓는 의미가 생기고, 복잡한 패턴을 학습할 수 있게 됩니다.

주요 활성화 함수로는 시그모이드(Sigmoid), 하이퍼볼릭 탄젠트(tanh), ReLU(Rectified Linear Unit) 등이 있는데, 그중 **시그모이드**와 **ReLU**에 대해 간단히 짚고 넘어가고자 합니다.

### 시그모이드(Sigmoid) 함수

시그모이드 함수는 신경망 초기에 널리 사용되었습니다. 계단 함수처럼 0과 1 사이의 값을 출력하지만, 이름처럼 'S자' 형태의 부드러운 곡선 형태를 가집니다.

$$f(x) = \frac{1}{1 + e^{-x}}$$

이 '부드러움'이 핵심입니다. 출력이 0.1, 0.73, 0.92…처럼 연속적으로 변하기 때문에, 신경망이 학습을 할 때 미세한 변화를 감지하고 가중치를 조절할 수 있게 됩니다. (이는 4부에서 다룰 '경사 하강법'과 직결됩니다.)

```python
import numpy as np
import matplotlib.pyplot as plt

def sigmoid(x):
    return 1 / (1 + np.exp(-x))    

x = np.arange(-5.0, 5.0, 0.1)
y = sigmoid(x)
plt.plot(x, y)
plt.ylim(-0.1, 1.1)
plt.show()
```

### ReLU(Rectified Linear Unit) 함수

시그모이드 함수는 훌륭했으나 **Vanishing Gradient(기울기 소실)**라는 고질적인 문제(신경망이 깊어질수록 학습 신호가 약해지는 현상)가 있습니다.

**ReLU(렐루)** 함수는 이 문제를 해결한 대표적인 활성화 함수입니다. 작동 방식은 매우 단순합니다.

$$y = \max(0, x)$$

* 입력이 0보다 크면, 입력을 그대로 출력한다. ($y=x$)
* 입력이 0보다 작거나 같으면, 0을 출력한다. ($y=0$)

이 단순함 덕분에 계산 속도가 빠르고 시그모이드의 기울기 소실 문제도 발생하지 않아 깊은 신경망을 효과적으로 학습시킬 수 있게 되었습니다.

```python
import numpy as np
import matplotlib.pyplot as plt

def relu(x):
    return np.maximum(0, x)

x = np.arange(-5.0, 5.0, 0.1)
y = relu(x)
plt.plot(x, y)
plt.ylim(-1.0, 5.5)
plt.show()
```

> **Dying ReLU 문제와 Leaky ReLU**
>
> ReLU는 강력한 활성화 함수지만, 만약 입력값이 지속적으로 0보다 작게 들어오면 해당 뉴런은 계속 0만 출력하게 되고, 그에 따라 (나중에 살펴볼) 역전파 시 기울기도 0이 되어 가중치가 더 이상 업데이트되지 않는 일명 **'Dying ReLU'** 현상이 발생할 수 있습니다.
>
> 이를 해결하기 위해 입력값이 0보다 작을 때 0 대신 $0.01 \times x$처럼 아주 작은 음의 기울기를 갖는 **Leaky ReLU** 같은 변형 함수들이 존재하는데, 이 내용은 추후 학습 최적화 포스트에서 다룰 예정입니다.

## 3층 신경망 구현 (순전파)

이제 활성화 함수를 이용해 신경망을 직접 만들어 보겠습니다. 입력부터 출력까지 데이터가 각 층을 순서대로 통과하는 이 과정을 **순전파(Forward Propagation)**라고 부릅니다.

여기서는 2개의 입력 뉴런, 3개의 뉴런을 가진 1번 은닉층, 2개의 뉴런을 가진 2번 은닉층, 그리고 2개의 출력 뉴런을 가진 3층 신경망을 구현합니다.

신경망 계산의 핵심은 **행렬의 곱(Dot Product)**입니다. NumPy 라이브러리의 `np.dot()`을 사용하면 한 층의 모든 뉴런 계산을 한 번에 처리할 수 있습니다.

### 1단계: 0층(입력층) → 1층(은닉층)

입력 $X$에 가중치 $W1$을 곱하고 편향 $B1$을 더해, 첫 번째 은닉층의 입력값 $A1$을 만듭니다.

$$A^{(1)} = X \cdot W^{(1)} + B^{(1)}$$

그리고 $A1$에 활성화 함수(여기서는 시그모이드)를 적용해 최종 출력 $Z1$을 얻습니다.

```python
X = np.array([1.0, 0.5])
W1 = np.array([[0.1, 0.3, 0.5], [0.2, 0.4, 0.6]])
B1 = np.array([0.1, 0.2, 0.3])

A1 = np.dot(X, W1) + B1
Z1 = sigmoid(A1)  # 활성화 함수 적용

print("A1(가중치 합): ", A1)  # [0.3 0.7 1.1]
print("Z1(1층 출력): ", Z1)   # [0.574... 0.668... 0.750...]
```

### 2단계: 1층(은닉층) → 2층(은닉층)

1층의 출력 $Z1$이 2층의 입력이 됩니다. 같은 과정을 반복합니다.

$$A^{(2)} = Z^{(1)} \cdot W^{(2)} + B^{(2)}$$

```python
W2 = np.array([[0.1, 0.4], [0.2, 0.5], [0.3, 0.6]])
B2 = np.array([0.1, 0.2])

A2 = np.dot(Z1, W2) + B2
Z2 = sigmoid(A2)  # 활성화 함수 적용

print("A2(가중치 합): ", A2)  # [0.516... 1.214...]
print("Z2(2층 출력): ", Z2)   # [0.626... 0.771...]
```

### 3단계: 2층(은닉층) → 3층(출력층)

드디어 마지막 층입니다. 2층의 출력 $Z2$를 입력으로 받아 최종 출력값 $A3$를 계산합니다.

$$A^{(3)} = Z^{(2)} \cdot W^{(3)} + B^{(3)}$$

```python
W3 = np.array([[0.2, 0.5], [0.3, 0.6]])
B3 = np.array([0.1, 0.2])

A3 = np.dot(Z2, W3) + B3
print("A3(최종 출력, raw): ", A3)  # [0.456... 0.975...]
```

> **출력층의 활성화 함수**
>
> 1, 2단계와 달리 최종 출력을 계산하는 3단계에서는 활성화 함수를 바로 적용하지 않고 $A3$ 값을 그대로 두었습니다. 출력층의 역할이 은닉층과 다르기 때문입니다.
>
> * **은닉층:** 정보의 비선형적 특징을 추출하는 것이 목적 → 시그모이드, ReLU 등 사용
> * **출력층:** 최종 결과를 우리가 원하는 형태(확률, 값 등)로 변환하는 것이 목적 → 문제 유형에 맞는 함수를 별도로 선택해서 적용
>
> 우리는 분류(Classification) 문제를 풀고 싶으므로 이 $A3$ 값을 '확률'로 바꿔주는 특별한 함수인 **소프트맥스(Softmax)**를 연결할 것입니다.

## 출력층 설계: 소프트맥스로 '확률' 만들기

앞서 계산한 $A3$ 값 `[0.456..., 0.975...]`는 신경망이 계산한 날것의 점수(로짓)인데, 이 점수는 범위가 정해져 있지 않아 직관적으로 해석하기 어렵습니다. 그래서 **소프트맥스(Softmax)**를 출력층의 활성화 함수로 사용합니다.

소프트맥스는 최종 점수들을 0에서 1 사이의 값으로 변환하고, **모든 출력의 합이 1이 되도록** 정규화(Normalize)합니다. 덕분에 우리는 이 결과를 '확률'로 해석할 수 있습니다.

$$y_k = \frac{e^{a_k}}{\sum_{i=1}^{n} e^{a_i}}$$

```python
def softmax(a):
    c = np.max(a)  # 오버플로(Overflow) 방지를 위해 뺄 로짓 최댓값
    exp_a = np.exp(a - c)
    sum_exp_a = np.sum(exp_a)
    y = exp_a / sum_exp_a
    return y

Y = softmax(A3)
print("Y(최종 확률): ", Y)  # [0.373... 0.626...]
```

`[0.373..., 0.626...]`이라는 결과는 신경망이 이 입력을 "37.3…%의 확률로 0번 클래스이고, 62.6…%의 확률로 1번 클래스일 것"으로 예측했다는 의미입니다.

소프트맥스에 관한 조금 더 자세한 내용은 [소프트맥스와 교차 엔트로피 손실](/posts/softmax-cross-entropy-loss/) 포스트를 참고해 주세요.
